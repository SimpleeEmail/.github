# Simplee.email

Simplee.email is a serverless, AWS-native email platform.

It provides a modern alternative to traditional mail servers by using:

- Amazon SES
- AWS Lambda
- DynamoDB
- S3
- CloudFront

No EC2. No Postfix. No legacy mail stack.

---

## What Is This?

Simplee.email is:

- Fully open-source core
- Bring-your-own-domain ready
- Designed for AWS-native builders
- Deployable via Infrastructure as Code

You can:
- Self-host in your own AWS account
- Or use our managed hosted version

---

## Architecture Philosophy

- Event-driven
- Multi-tenant capable
- Infrastructure as Code
- Minimal surface area
- Cost transparent
- Security first
